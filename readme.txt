A block that will collate all images in posts with a given tag into a gallery.

== Description ==

Provides a block that can be used in the Wordpress Block Editor.

It has only a single configuration option, the tag. Given, it will
find and collate all images (along with alt text, etc) present in
posts with that tag.

This plugin was commissioned by [Tikaka](https://tikaka.art)!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tag-gallery` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
